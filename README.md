# As aventuras de Adora

Jogo desenvolvido utilizando a biblioteca P5 (JavaScript).


## Motivação

Fiz esse jogo para testar meus conhecimentos adquiridos na #ImersaoGameDev da Alura.
Agradeço demais aos professores, Juliana e Guilherme, pelos ensinamentos. 

Definitivamente, um dos códigos mais divertidos que eu já fiz.

Me inspirei em She-Ra - As Princesas do Poder para criar os sprites e o cenários.

Obs: infelizmente fui eu mesma quem fez as artes e, por isso, a qualidade não está lá tão boa.


## Pontos de melhoria

Pretendo melhorar os seguintes pontos em um futuro indeterminado:
* Implementar um highscore
* Implementar mais duas fases: Florestas Assombradas e Lua Clara
* Melhorar as artes do jogo
